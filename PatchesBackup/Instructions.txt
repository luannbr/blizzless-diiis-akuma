1. Install PostgreSQL 9.5.25

⚠️ Important: You must install PostgreSQL version 9.5.25. Newer versions may NOT work correctly with this server.

Download PostgreSQL 9.5.25 from an official archive or trusted mirror.

Run the installer.

During installation:

Port: leave default (5432)

Locale: leave default

Password:

Set the password for the postgres user as: postgres

Write this password down, you will need it later

Finish the installation.

After installation, PostgreSQL will start automatically.

2. Open pgAdmin

Open pgAdmin (installed with PostgreSQL).

When prompted, log in using:

Username: postgres

Password: the password you set during installation

You should now see your PostgreSQL server in the left panel.

3. Create Required Databases

You must create two databases:

diiis

worlds

Create the databases

In pgAdmin, expand:

Servers → PostgreSQL → Databases


Right-click Databases

Click Create → Database…

Create the first database:

Database name: diiis

Click Save

Repeat the same steps for the second database:

Database name: worlds

Click Save

You should now see both databases listed.

4. Restore worlds.backup to the Worlds Database

Now you need to import the world data.

Restore the backup

In pgAdmin, right-click the worlds database

Click Restore…

In the restore window:

Format: Custom or Tar

Filename: select worlds.backup

Click Restore

Wait until the process completes (this may take a few minutes)

If no errors appear, the restore was successful.
